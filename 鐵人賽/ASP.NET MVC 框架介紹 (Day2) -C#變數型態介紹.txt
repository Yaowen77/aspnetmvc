註解1: = 是 取代的意思 表示 右邊取代左邊 

ex:
int a = 1;

int b = 2;

a = b;

此時結果 a 是 2



註解2: == 是 相同的意思 表示 左右兩邊是否相等

ex:

int a = 1;

int b = 1;


if(a == b){ //2者數值相同 進入判斷式
   Console.WriteLine(a);
}
